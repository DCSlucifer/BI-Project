# Business Intelligence Final Project - Superstore Report Outline

This outline defines the recommended structure of the final report and aligns the written explanation with the current implementation. The project loads the CSV source directly into the main SQL Server database, `SuperstoreDW`. The table `dbo.stg_Orders` is an internal landing table within `SuperstoreDW`, not a separate temporary database. The analytical warehouse is modeled as a star schema, with `FactSales` as the central fact table.

## 5. Data Mining

### 5.1 Decision Tree - Loss-Risk Rule Discovery

- **Objective:** Classify each order line as `Profitable` or `Loss` using predictors such as discount, category, sub-category, segment, region, ship mode, quantity, and sales.
- **Rationale:** A Decision Tree is appropriate because it produces interpretable rules, not only predictive scores.
- **Validation evidence to report:**
  - Baseline accuracy: 80.63%.
  - Decision Tree test accuracy: 94.40%.
  - Macro F1-score: 90.33%.
  - 5-fold cross-validation accuracy: 94.44% +/- 0.20%.
  - Random Forest benchmark accuracy: 94.28%.
- **Validated rule:** `Discount > 0.25 AND Category != Technology` predicts `Loss` with test support of 318 and precision of 99.37%.
- **Outputs to include:** `class_balance.png`, `confusion_matrix.png`, `feature_importance.png`, `decision_tree.png`, `loss_rules.png`, `decision_rules.csv`, `loss_rules.csv`, and `model_summary.txt`.

### 5.2 K-Means RFM Customer Segmentation

- **Objective:** Segment customers according to `Recency`, `Frequency`, and `Monetary` behavior.
- **Preprocessing:** Aggregate transactions by customer and standardize RFM values with `StandardScaler`.
- **Cluster selection:** The number of clusters is kept at `k = 4`, supported by the Elbow Method and Silhouette analysis.
- **Stable cluster labels:** The segment names and colors are retained for consistency with previous notes:
  - Cluster 0: `Trung thanh / Active`.
  - Cluster 1: `Pho thong / Trung binh`.
  - Cluster 2: `VIP / Champions`.
  - Cluster 3: `Roi bo / Lost`.
- **Outputs to include:** `elbow_silhouette.png`, `segments.png`, `rfm_profile.png`, and `rfm_customers.csv`.
- **Methodological distinction:** K-Means is an unsupervised segmentation method, while the Decision Tree is a supervised rule-discovery method.
